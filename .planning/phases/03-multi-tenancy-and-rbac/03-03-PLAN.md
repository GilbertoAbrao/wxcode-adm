1. Add invitation functions to `backend/src/wxcode_adm/tenants/service.py`:

The module-level `invitation_serializer` should already exist from Plan 03-02. If not, add it:
```python
invitation_serializer = URLSafeTimedSerializer(
    settings.JWT_PRIVATE_KEY.get_secret_value(),
    salt="tenant-invitation",
)
```

**generate_invitation_token(email: str, tenant_id: str) -> str:**
- Returns `invitation_serializer.dumps({"email": email, "tenant_id": tenant_id})`

**verify_invitation_token(token: str) -> dict:**
- Calls `invitation_serializer.loads(token, max_age=7 * 24 * 3600)` (7 days)
- On SignatureExpired: raise TokenExpiredError()
- On BadSignature: raise InvalidTokenError()
- Returns the payload dict

**invite_user(db: AsyncSession, redis: Redis, tenant: Tenant, membership: TenantMembership, body: InviteRequest) -> Invitation:**
- Guard: if membership.role.level < MemberRole.ADMIN.level, raise InsufficientRoleError (shouldn't happen if require_role is used on the route, but defense in depth)
- Check if email is already a member of this tenant: query TenantMembership joined with User where User.email == body.email AND TenantMembership.tenant_id == tenant.id. If found, raise AlreadyMemberError()
- Check for existing active (non-expired, non-accepted) invitation: query Invitation where email == body.email AND tenant_id == tenant.id AND accepted_at IS NULL AND expires_at > now(). If found, raise InvitationAlreadyExistsError()
- Generate invitation token: token = generate_invitation_token(body.email, str(tenant.id))
- token_hash = hashlib.sha256(token.encode()).hexdigest()
- Create Invitation record: email=body.email, tenant_id=tenant.id, role=MemberRole(body.role), token_hash=token_hash, invited_by_id=membership.user_id, expires_at=datetime.now(tz=timezone.utc) + timedelta(days=7), billing_access=body.billing_access
- db.add(invitation), db.flush()
- Build invitation link: f"{settings.ALLOWED_ORIGINS[0]}/invitations/accept?token={token}"
- Enqueue arq job: enqueue_job("send_invitation_email", body.email, tenant.name, link, body.role)
- Return the invitation record

**accept_invitation(db: AsyncSession, user: User, token: str) -> TenantMembership:**
- Verify the token to get payload (email, tenant_id)
- Verify the token's email matches the authenticated user's email. If not, raise ForbiddenError("INVITATION_EMAIL_MISMATCH", "This invitation was sent to a different email address")
- Compute token_hash = hashlib.sha256(token.encode()).hexdigest()
- Look up Invitation by token_hash — if not found, raise InvalidTokenError()
- If invitation.accepted_at is not None: raise ConflictError("INVITATION_ALREADY_ACCEPTED", "This invitation has already been accepted")
- Check expiry: if invitation.expires_at < now(), raise TokenExpiredError()
- Check if user is already a member: query TenantMembership where user_id == user.id AND tenant_id == invitation.tenant_id. If found, raise AlreadyMemberError()
- Create TenantMembership: user_id=user.id, tenant_id=invitation.tenant_id, role=invitation.role, billing_access=invitation.billing_access (from the invitation), invited_by_id=invitation.invited_by_id
- Mark invitation.accepted_at = datetime.now(tz=timezone.utc)
- db.add(membership), db.flush()
- Return membership

**list_invitations(db: AsyncSession, tenant_id: uuid.UUID) -> list[Invitation]:**
- Query all Invitation records for this tenant_id where accepted_at IS NULL, ordered by created_at desc
- Return the list (includes both active and expired -- frontend can filter)

**cancel_invitation(db: AsyncSession, tenant_id: uuid.UUID, invitation_id: uuid.UUID) -> None:**
- Find Invitation by id AND tenant_id — if not found, raise NotFoundError("INVITATION_NOT_FOUND", "Invitation not found")
- If already accepted, raise ConflictError("INVITATION_ALREADY_ACCEPTED", "Cannot cancel an accepted invitation")
- Delete the invitation record
