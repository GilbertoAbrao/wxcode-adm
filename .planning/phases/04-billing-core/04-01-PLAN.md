    2. **billing/service.py**: Create plan CRUD + Stripe sync functions:
       - Add `from __future__ import annotations` for Python 3.9 compat.
       - Import `stripe_client` from `billing/stripe_client.py`.

       **`async def create_plan(db, body: CreatePlanRequest) -> Plan`**:
       1. Check slug uniqueness (SELECT Plan WHERE slug=body.slug). Raise ConflictError(PLAN_SLUG_EXISTS) if taken.
       2. Create Plan ORM instance from body, flush to get ID.
       3. Sync to Stripe (all in try/except — Stripe failure should NOT block plan creation, log warning):
          a. Create Billing Meter: `stripe_client.billing.meters.create_async(params={...})` with event_name=f"wxcode_tokens_{body.slug}", default_aggregation={"formula": "sum"}, customer_mapping={"event_payload_key": "stripe_customer_id", "type": "by_id"}, value_settings={"event_payload_key": "value"}.
          b. Create Product: `stripe_client.products.create_async(params={"name": body.name, "metadata": {"plan_id": str(plan.id)}})`
          c. Create flat-fee Price: `stripe_client.prices.create_async(params={...})` with product=product.id, currency="usd", unit_amount=body.monthly_fee_cents, recurring={"interval": "month", "usage_type": "licensed"}.
          d. Create overage Price: `stripe_client.prices.create_async(params={...})` with product=product.id, currency="usd", unit_amount_decimal=str(body.overage_rate_cents_per_token), billing_scheme="per_unit", recurring={"interval": "month", "usage_type": "metered", "meter": meter.id}.
          e. Update Plan with stripe_product_id, stripe_price_id, stripe_meter_id, stripe_overage_price_id.
       4. Return plan.

       **`async def update_plan(db, plan_id: UUID, body: UpdatePlanRequest) -> Plan`**:
       1. Load Plan by ID. Raise NotFoundError(PLAN_NOT_FOUND) if missing.
       2. Apply non-None fields from body to plan.
       3. If monthly_fee_cents or overage_rate_cents_per_token changed AND plan has stripe_product_id:
          a. Archive old Stripe Price: `stripe_client.prices.update_async(plan.stripe_price_id, params={"active": False})`.
          b. Create new Price with updated amount.
          c. Update plan.stripe_price_id (and/or stripe_overage_price_id).
          Wrap Stripe calls in try/except, log warning on failure.
       4. Return plan.

       **`async def delete_plan(db, plan_id: UUID) -> Plan`**:
       1. Load Plan by ID. Raise NotFoundError(PLAN_NOT_FOUND) if missing.
       2. Set is_active=False (soft delete — cannot hard delete if subscriptions reference it).
       3. If plan has stripe_product_id, archive the Stripe Product: `stripe_client.products.update_async(plan.stripe_product_id, params={"active": False})`. Wrap in try/except.
       4. Return plan.

       **`async def list_plans(db, include_inactive: bool = False) -> list[Plan]`**:
       1. Query Plan. If not include_inactive, filter WHERE is_active=True.
       2. Order by monthly_fee_cents ASC (free first, then ascending price).
       3. Return list.

       **`async def get_plan(db, plan_id: UUID) -> Plan`**:
       1. Load Plan by ID. Raise NotFoundError(PLAN_NOT_FOUND) if missing.
       2. Return plan.
