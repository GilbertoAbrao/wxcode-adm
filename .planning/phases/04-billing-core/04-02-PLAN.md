    2. **billing/service.py**: Add checkout session creation (append to existing):

       **`async def create_checkout_session(db: AsyncSession, tenant_id: UUID, plan_id: UUID) -> tuple[str, str]`**:
       1. Load the TenantSubscription for this tenant. Raise PaymentRequiredError("No subscription record found") if missing.
       2. Load the target Plan by plan_id. Raise NotFoundError(PLAN_NOT_FOUND) if missing or not active.
       3. Guard: if plan.monthly_fee_cents == 0, raise ConflictError(CANNOT_CHECKOUT_FREE, "Cannot checkout for the free plan — use upgrade instead").
       4. Guard: if subscription already has stripe_subscription_id and status is ACTIVE, raise ConflictError(ALREADY_SUBSCRIBED, "Already subscribed — use Customer Portal to manage").
       5. If stripe_customer_id is None on the subscription (Stripe Customer creation failed during onboarding), create one now:
          - Load tenant name from Tenant table, load user email from the subscription context.
          - Call create_stripe_customer and update subscription.stripe_customer_id.
       6. Determine line_items:
          - `{"price": plan.stripe_price_id, "quantity": 1}` (flat fee)
          - `{"price": plan.stripe_overage_price_id}` (metered overage — no quantity for metered prices, Stripe ignores it but do NOT pass quantity for clarity)
          - ONLY include overage line item if plan.stripe_overage_price_id is not None.
       7. Create checkout session:
          ```python
          session = await stripe_client.checkout.sessions.create_async(params={
              "customer": subscription.stripe_customer_id,
              "mode": "subscription",
              "line_items": line_items,
              "success_url": f"{settings.FRONTEND_URL}/billing?session_id={{CHECKOUT_SESSION_ID}}",
              "cancel_url": f"{settings.FRONTEND_URL}/billing?canceled=true",
              "metadata": {"tenant_id": str(tenant_id), "plan_id": str(plan_id)},
              "subscription_data": {
                  "metadata": {"tenant_id": str(tenant_id), "plan_id": str(plan_id)},
              },
          })
          ```
       8. Return (session.url, session.id).

       Per user decision: post-checkout redirects to /billing?session_id=... (Claude's Discretion).
