2. **src/lib/utils.ts** -- Port from wxcode frontend (identical):
   ```typescript
   import { type ClassValue, clsx } from "clsx"
   import { twMerge } from "tailwind-merge"
   export function cn(...inputs: ClassValue[]) {
     return twMerge(clsx(inputs))
   }
   ```

3. **src/app/globals.css** -- Create a MINIMAL placeholder for now (full theme ported in Plan 02):
   ```css
   @import "tailwindcss";
   @import "tw-animate-css";
   @layer base {
     * { @apply border-border outline-ring/50; }
     body { @apply bg-background text-foreground; }
   }
   ```
   Include only the `@theme inline` block with the shadcn CSS variables needed for the base shadcn components to render (--background, --foreground, --primary, etc.). Use Obsidian dark defaults for the `.dark` class. This is a minimal bootstrap -- Plan 02 adds the full Obsidian token set.

4. **src/app/layout.tsx** -- Create root layout:
   - Import Geist and Geist_Mono fonts from `next/font/google`
   - Set `<html lang="en" className="dark" suppressHydrationWarning>`
   - Apply font variables and `antialiased bg-background text-foreground` to body
   - Metadata: title "wxCode Admin", description "Identity, tenancy, and billing management for wxCode"
   - No QueryProvider yet (added in Plan 03)
