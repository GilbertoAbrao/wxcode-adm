3. **audit/models.py** — Create AuditLog model following the EXACT schema from research:
   ```python
   class AuditLog(Base):
       __tablename__ = "audit_logs"
   ```

   CRITICAL: Do NOT use TimestampMixin. AuditLog has NO `updated_at` column. Declare `id` (UUID, gen_random_uuid server_default) and `created_at` (DateTime with timezone, server_default now()) manually.

   Columns:
   - `id`: UUID primary key, default uuid4, server_default gen_random_uuid()
   - `created_at`: DateTime(timezone=True), server_default now(), nullable=False
   - `actor_id`: UUID, ForeignKey("users.id", ondelete="SET NULL"), nullable=True, index=True
   - `tenant_id`: UUID, ForeignKey("tenants.id", ondelete="SET NULL"), nullable=True, index=True
   - `action`: String(100), nullable=False, index=True (e.g., "login", "invite_user", "update_plan")
   - `resource_type`: String(100), nullable=False (e.g., "user", "tenant", "invitation", "plan")
   - `resource_id`: String(255), nullable=True (stringified UUID of affected resource)
   - `ip_address`: String(45), nullable=True (IPv4 or IPv6)
   - `details`: JSONB, nullable=False, server_default "'{}'::jsonb"

   Use `from sqlalchemy.dialects.postgresql import JSONB`.
   Add `from __future__ import annotations` for Python 3.9.6 compat (matching existing pattern).

4. **audit/service.py** — Create two functions:

   a) `write_audit(db: AsyncSession, actor_id, action, resource_type, resource_id, tenant_id, ip_address, details)`:
      - Creates AuditLog instance and adds to session
      - Does NOT commit — caller's session commit handles it
      - `details` defaults to empty dict if None
      - All params except `db` and `action` and `resource_type` are optional

   b) `purge_old_audit_logs(ctx: dict) -> int`:
      - arq cron job function (receives ctx with session_maker)
      - Calculates cutoff = now() - timedelta(days=settings.AUDIT_LOG_RETENTION_DAYS)
      - Executes `DELETE FROM audit_logs WHERE created_at < cutoff`
      - Returns rowcount
      - Uses its own session from ctx["session_maker"] (following existing worker pattern)

5. **tasks/worker.py** — Register the cron job:
   - Import `from arq import cron` at top
   - Import `from wxcode_adm.audit.service import purge_old_audit_logs`
   - Add `cron_jobs` to WorkerSettings class: `cron_jobs = [cron(purge_old_audit_logs, hour=2, minute=0)]`
   - This runs daily at 2:00 AM UTC
