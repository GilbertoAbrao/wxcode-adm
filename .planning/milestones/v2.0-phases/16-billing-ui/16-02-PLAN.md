5. Implementation approach for conditional polling — use the `refetchInterval` option on useSubscription:

```typescript
const isPolling = !!sessionId && subscription?.status === "free";

const { data: subscription, ... } = useSubscription(tenantId, {
  refetchInterval: isPolling ? 2000 : false,
});
```

To support this, modify `useSubscription` in useBilling.ts to accept an optional `options` parameter:

```typescript
export function useSubscription(
  tenantId: string | undefined,
  options?: { refetchInterval?: number | false }
) {
  return useQuery<Subscription, Error>({
    queryKey: BILLING_QUERY_KEYS.subscription(tenantId!),
    queryFn: () => apiClient<Subscription>("/billing/subscription", {
      ...tenantHeaders(tenantId!),
    }),
    enabled: !!tenantId,
    staleTime: 30_000,
    refetchInterval: options?.refetchInterval,
  });
}
```

6. When subscription status changes from "free" to any other value while sessionId is present:
   - Set `checkoutComplete = true`
   - Use `router.replace("/billing", { scroll: false })` to remove session_id from URL without page reload
   - Show a success banner: "Subscription activated! You are now on the {plan.name} plan." (bg-emerald-400/10 text-emerald-400, auto-dismiss after 5 seconds)

7. Track polling with a useEffect:

```typescript
useEffect(() => {
  if (!sessionId) return;
  if (subscription && subscription.status !== "free") {
    // Payment processed — clean up
    setCheckoutComplete(true);
    router.replace("/billing", { scroll: false });
    return;
  }
  // Count polls via subscription refetch cycle

```

8. For the "max polls exceeded" case: after 20 seconds of polling with no change, stop the refetchInterval by setting a `pollTimedOut` state:

```typescript
const [pollTimedOut, setPollTimedOut] = useState(false);

useEffect(() => {
  if (!sessionId || pollTimedOut) return;
  const timeout = setTimeout(() => setPollTimedOut(true), 20_000);
  return () => clearTimeout(timeout);

// In the query:
const isPolling = !!sessionId && subscription?.status === "free" && !pollTimedOut;
```

When `pollTimedOut` is true, show: "Your payment is being processed. The subscription may take a moment to activate. Please refresh the page shortly."
