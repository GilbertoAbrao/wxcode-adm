   - **Desktop (lg+):** Fixed left sidebar, width 256px (w-64), full height, bg-sidebar (--obsidian-900 in dark), border-r border-sidebar-border
   - **Mobile (<lg):** Hidden by default. Hamburger button (Menu icon) in a fixed top bar triggers a slide-in overlay sidebar with backdrop
   - **Sidebar content:**
     - Logo area at top: "wxCode" text with blue accent, or text "WX" in a small rounded square + "wxCode Admin"

       - Billing (CreditCard icon) -- links to /billing (Phase 16)

     - Hover state: bg-sidebar-accent/50
     - Bottom section: settings gear icon, user avatar placeholder

   - Mobile state managed with `useState` for open/close
   - Close sidebar on route change (useEffect watching pathname)
   - Smooth transition for sidebar slide-in on mobile (translate-x, duration-300)
   - Backdrop: fixed overlay with bg-black/50, click to close
