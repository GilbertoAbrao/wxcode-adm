- `signup`: After `await service.signup(db, redis, body)` succeeds:

  ```python
  # After the login call succeeds, we need the user_id. The service already looked it up.
  # Approach: add an import and quick lookup, OR modify the login endpoint to also return user context.
  # Simplest: query user by email for the audit (one extra lightweight query).
  from sqlalchemy import select
  from wxcode_adm.auth.models import User
  user_result = await db.execute(select(User.id).where(User.email == body.email))
  user_row = user_result.scalar_one_or_none()

      await write_audit(db, actor_id=user_row, action="login", resource_type="session", ip_address=request.client.host if request.client else None)
  ```
