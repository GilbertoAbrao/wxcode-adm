    1. **billing/dependencies.py**: Create 3 reusable FastAPI dependencies for plan enforcement,
       PLUS 2 private synchronous helpers that contain the core enforcement logic.

       The private helpers are required so that integration tests can call the enforcement path
       directly (without FastAPI Depends wiring) and assert it raises the correct exceptions.
       Plan 05 tests call `_enforce_token_quota` and `_enforce_active_subscription` directly.

       ```python
       """
       Plan enforcement dependencies for wxcode-adm.

       These dependencies are wired into endpoints that proxy to the wxcode engine
       or manage tenant resources with plan-enforced limits.

       Enforcement rules (from CONTEXT.md locked decisions):
       - Paid plans: overage billing (never block a paying customer on token quota)
       - Free tier: hard block at token quota (HTTP 402 with upgrade prompt)
       - Member cap: hard limit on ALL plans (HTTP 402 when at/above cap)
       - Past_due/canceled: block wxcode engine access (HTTP 402)
       """
       from __future__ import annotations

       from fastapi import Depends, Response
       from sqlalchemy import func, select
       from sqlalchemy.ext.asyncio import AsyncSession

       from wxcode_adm.billing.exceptions import MemberLimitError, PaymentRequiredError, QuotaExceededError
       from wxcode_adm.billing.models import Plan, SubscriptionStatus, TenantSubscription
       from wxcode_adm.dependencies import get_session
       from wxcode_adm.tenants.dependencies import get_tenant_context
       from wxcode_adm.tenants.models import Tenant, TenantMembership
       ```

       **Private helper — `_enforce_active_subscription(subscription: TenantSubscription) -> None`**:
       - Pure synchronous function, no DB calls, no Depends.
       - If subscription.status in (SubscriptionStatus.PAST_DUE, SubscriptionStatus.CANCELED):
         raise PaymentRequiredError("Subscription payment required. Visit billing settings to resolve.")
       - Called by `require_active_subscription` dependency. Also called directly in tests.

       **Private helper — `_enforce_token_quota(plan: Plan, subscription: TenantSubscription) -> None`**:
       - Pure synchronous function, no DB calls, no Depends.
       - Free tier hard block: if plan.monthly_fee_cents == 0 AND plan.token_quota > 0
         AND subscription.tokens_used_this_period >= plan.token_quota:
         raise QuotaExceededError(f"Free plan token quota ({plan.token_quota}) exhausted. Upgrade to continue.")
       - Paid tier: do nothing (overage billing, never block per locked decision).
       - Called by `check_token_quota` dependency. Also called directly in tests.

       **`async def require_active_subscription(...) -> tuple[Tenant, TenantMembership, TenantSubscription]`**:
       - Depends: get_tenant_context, get_session
       - Query TenantSubscription WHERE tenant_id = tenant.id
       - If None: raise PaymentRequiredError with message "Subscription payment required..."
       - Call `_enforce_active_subscription(subscription)` — raises if past_due/canceled.
       - Status FREE is allowed (free tier is a valid subscription state).
       - Return (tenant, membership, subscription).

       **`async def check_token_quota(response: Response, ...) -> tuple[Tenant, TenantMembership, TenantSubscription]`**:
       - Depends: require_active_subscription, get_session
       - Load Plan from subscription.plan_id.
       - Call `_enforce_token_quota(plan, subscription)` — raises for free tier at quota.
       - Calculate usage_pct = subscription.tokens_used_this_period / plan.token_quota
         (guard division by zero if token_quota is 0).
       - **Warning headers** (per CONTEXT: "Warning headers at 80% and 100% of token quota"):
         If usage_pct >= 1.0: `response.headers["X-Quota-Warning"] = "QUOTA_REACHED"`,
           `response.headers["X-Quota-Usage"] = f"{subscription.tokens_used_this_period}/{plan.token_quota}"`.
         Elif usage_pct >= 0.8: `response.headers["X-Quota-Warning"] = "QUOTA_WARNING_80PCT"`,
           `response.headers["X-Quota-Usage"] = f"{subscription.tokens_used_this_period}/{plan.token_quota}"`.
       - Return (tenant, membership, subscription).

       Note: The `Response` parameter is injected by FastAPI — it allows dependencies to set response headers. This is the documented pattern for dependencies that need to modify the response.

       **`async def check_member_cap(...) -> tuple[Tenant, TenantMembership]`**:
       - Depends: get_tenant_context, get_session
       - Query TenantSubscription for this tenant. If None, return (tenant, membership) — no enforcement without a subscription.
       - Load Plan from subscription.plan_id.
       - If plan.member_cap <= 0 (unlimited, -1 convention): return (tenant, membership).
       - Count current members: `SELECT COUNT(*) FROM tenant_memberships WHERE tenant_id = tenant.id`.
       - If member_count >= plan.member_cap: raise MemberLimitError(message=f"Plan member limit ({plan.member_cap}) reached. Upgrade to invite more members.").
       - Return (tenant, membership).
