1. **auth/jwt.py** — Create JWT utility module:
   - `import jwt` (PyJWT)
   - `import uuid`
   - `from datetime import datetime, timedelta, timezone`
   - `from wxcode_adm.config import settings`
   - Function `create_access_token(user_id: str, extra_claims: dict | None = None) -> str`:
     - Build payload: `sub=user_id`, `iat=now`, `exp=now + timedelta(hours=settings.ACCESS_TOKEN_TTL_HOURS)`, `jti=str(uuid.uuid4())`, extra_claims merged if provided
     - Sign with `jwt.encode(payload, settings.JWT_PRIVATE_KEY.get_secret_value(), algorithm="RS256", headers={"kid": settings.JWT_KID})`
     - Return the encoded token string
   - Function `decode_access_token(token: str) -> dict`:
     - Call `jwt.decode(token, settings.JWT_PUBLIC_KEY.get_secret_value(), algorithms=["RS256"])`
     - On `jwt.ExpiredSignatureError` raise `TokenExpiredError()`
     - On `jwt.InvalidTokenError` raise `InvalidTokenError()`
     - Return payload dict
   - IMPORTANT: `kid` is set in the `headers` param of `jwt.encode()`, NOT in the payload

2. **auth/jwks.py** — Create JWKS response builder:
   - `import json`
   - `from jwt.algorithms import RSAAlgorithm`
   - `from cryptography.hazmat.primitives.serialization import load_pem_public_key`
   - Function `build_jwks_response(public_key_pem: str, kid: str = "v1") -> dict`:
     - Load PEM: `pub_key = load_pem_public_key(public_key_pem.encode())`
     - Convert: `jwk_str = RSAAlgorithm.to_jwk(pub_key)`
     - Parse: `jwk_dict = json.loads(jwk_str)`
     - Add standard fields: `jwk_dict.update({"use": "sig", "alg": "RS256", "kid": kid})`
     - Return `{"keys": [jwk_dict]}`
   - NOTE: Do NOT pass `backend=default_backend()` — it is deprecated in modern cryptography; `load_pem_public_key` without backend uses the default automatically

3. **auth/router.py** — Create the auth router with JWKS endpoint:
   - `router = APIRouter(tags=["auth"])`
   - `@router.get("/.well-known/jwks.json")` — NO prefix (this must be at the root path, not under /api/v1)
     - `async def jwks_endpoint() -> dict:`
     - Call `build_jwks_response(settings.JWT_PUBLIC_KEY.get_secret_value(), kid=settings.JWT_KID)`
     - Return the response dict
   - The JWKS endpoint is public (no auth required)

4. **main.py** — Register the auth router in `create_app()`:
   - Import: `from wxcode_adm.auth.router import router as auth_router`
   - Add: `app.include_router(auth_router)` — NO prefix for JWKS (root-level `.well-known` is a standard convention)
   - Place after the existing common_router registration
   - IMPORTANT: The auth_router is registered WITHOUT `settings.API_V1_PREFIX` because `/.well-known/jwks.json` is a standard path that must be at the root. Future auth endpoints (login, signup) will be added under `prefix=settings.API_V1_PREFIX + "/auth"` in Plan 02.
