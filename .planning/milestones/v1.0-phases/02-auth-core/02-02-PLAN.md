   - Note: EmailStr comes from pydantic, available since pydantic v2 without extra install

2. **auth/service.py** â€” Create auth service module with signup and verification logic:

   **OTP helper functions** (module-level, using Redis):
   - `async def create_verification_code(redis: Redis, user_id: str) -> str`:
     - Generate 6-digit code: `str(secrets.randbelow(900000) + 100000)`
     - Store in Redis: `auth:otp:{user_id}` with TTL 600 (10 minutes)
     - Reset attempts: `auth:otp:attempts:{user_id}` = "0" with TTL 600
     - Set cooldown: `auth:otp:cooldown:{user_id}` = "1" with TTL 60 (1 minute)
     - Return the code string

   - `async def verify_otp_code(redis: Redis, user_id: str, submitted: str) -> bool`:
     - Get stored code from `auth:otp:{user_id}` â€” return False if None (expired)
     - **Check code FIRST:** If submitted == stored: delete both keys (otp + attempts), return True
     - Increment attempts: `new_attempts = await redis.incr(f"auth:otp:attempts:{user_id}")`
     - If new_attempts >= 3: delete both keys (otp + attempts), return False (3rd failure = lockout, force resend)
     - Return False (1st or 2nd failure, code still valid)

   - `async def check_resend_cooldown(redis: Redis, user_id: str) -> bool`:
     - Check if `auth:otp:cooldown:{user_id}` exists
     - Return True if cooldown active, False if not

   **Service functions:**
   - `async def signup(db: AsyncSession, redis: Redis, body: SignupRequest) -> User`:
     - Check if email exists: `select(User).where(User.email == body.email)` â€” if found, raise `EmailAlreadyExistsError`
     - Hash password: `hash_password(body.password)`
     - Create User: `User(email=body.email, password_hash=hashed, email_verified=False)`
     - `db.add(user)` then `await db.flush()` to get the user.id
     - Create OTP: `code = await create_verification_code(redis, str(user.id))`
     - Enqueue email job: `pool = await get_arq_pool(); await pool.enqueue_job("send_verification_email", str(user.id), user.email, code); await pool.aclose()`
     - Return user

   - `async def verify_email(db: AsyncSession, redis: Redis, body: VerifyEmailRequest) -> None`:
     - Find user by email: `select(User).where(User.email == body.email)` â€” if not found, raise `InvalidCredentialsError()` (prevent enumeration)
     - If user.email_verified is True, return silently (idempotent)
     - Call `verify_otp_code(redis, str(user.id), body.code)` â€” if False, raise `AppError(error_code="AUTH_INVALID_CODE", message="Invalid or expired verification code", status_code=400)`
     - Set `user.email_verified = True`

   - `async def resend_verification(db: AsyncSession, redis: Redis, body: ResendVerificationRequest) -> None`:
     - Find user by email â€” if not found, return silently (prevent enumeration)
     - If user.email_verified is True, return silently
     - Check cooldown: if active, raise `AppError(error_code="AUTH_COOLDOWN", message="Please wait before requesting a new code", status_code=429)`
     - Create new code: `code = await create_verification_code(redis, str(user.id))`
     - Enqueue email job to send new code
