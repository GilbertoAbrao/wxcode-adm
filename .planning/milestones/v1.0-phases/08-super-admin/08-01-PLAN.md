    1. Create `backend/src/wxcode_adm/admin/service.py` with three functions:
       - `admin_login(db, redis, email, password, client_ip) -> dict`: Load User by email, verify `is_superuser=True`, verify password via `verify_password` from auth/service.py. Issue admin access token via `create_admin_access_token(str(user.id))`. Create RefreshToken row (reuse exact pattern from auth/service.py `_issue_tokens` for refresh token — secrets.token_urlsafe, TTL from settings.REFRESH_TOKEN_TTL_DAYS). Write audit log via `write_audit(db, action="admin_login", resource_type="user", actor_id=user.id, ip_address=client_ip)`. Return `{"access_token": ..., "refresh_token": ...}`. If user not found, not superuser, or wrong password, raise `InvalidCredentialsError()`.
       - `admin_refresh(db, redis, refresh_token_str) -> dict`: Find RefreshToken row by token value. If not found or expired, raise `InvalidTokenError()`. Delete old row, create new RefreshToken + new admin access token. Return same shape as login. Reuse single-session / replay detection pattern from auth/service.py refresh function.
       - `admin_logout(db, redis, refresh_token_str, access_token_jti) -> None`: Delete RefreshToken row. Blacklist access JTI via `blacklist_jti(redis, access_token_jti)`. Write audit: `admin_logout`.

    2. Create `backend/src/wxcode_adm/admin/router.py`:
       - `admin_router = APIRouter(prefix="/admin", tags=["Admin"])`.
       - Import `limiter` from `wxcode_adm.common.rate_limit`.
       - `POST /login`: Rate limited `@limiter.limit("10/minute")` (Claude's discretion: stricter than regular 5/min would mean less lenient, but 10/min is fine for admin). Accept `request: Request` as first param (slowapi requirement). Parse `AdminLoginRequest` body. IP allowlist check: if `settings.ADMIN_ALLOWED_IPS` is truthy, split by comma, strip whitespace, check `request.client.host` against the list; raise `ForbiddenError(error_code="IP_NOT_ALLOWED", message="Access denied from this IP address")` if not in list. Call `admin_login(db, redis, body.email, body.password, request.client.host)`. Return `AdminTokenResponse`.
       - `POST /refresh`: Accept `{"refresh_token": str}` body. Call `admin_refresh`. Return `AdminTokenResponse`.
       - `POST /logout`: Requires `require_admin` dependency to get current admin user. Extract JTI from admin token. Accept `{"refresh_token": str}` body. Call `admin_logout`. Return 200 with `{"message": "Logged out"}`.
       - Ensure `request: Request` is the first parameter on login endpoint (slowapi requirement from Phase 5 decision).
